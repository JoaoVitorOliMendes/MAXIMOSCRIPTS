#Script para a inserção de semanas no bd
COMO USAR
  - Instalar o pymssql (pip install pymssql)
  - Mudar conexão para o seu banco
  - Checar string de inserção (ORGID e SITEID estão setados como EAGLENA BEDFORD como default)
  - Rodar o app
  - Digitar um ano válido YYYY (CUIDADO! Não há checagem de input)
